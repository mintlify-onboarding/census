## 🏃‍♀️ Getting Started

1. 1.  
Navigate to the **Destinations** page in Census and click **New Destination**.

2. 2.  
Select **Userpilot** from the menu.

3. 3.  
Enter your **API Key**, which can be found in the Userpilot app under **Configure** \> **Environment**.

4. 4.  
If you have an Enterprise plan with Userpilot, enter your **Enterprise Base URL**. Otherwise, leave it blank.

![](/images/destinations/userpilot.png)

Copy your API key from the Userpilot app.
